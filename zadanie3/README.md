Do kompilacji trzeba użyć pythona3.
W celu poprawnej kompilacji należy użyć następujących komend:
1. sudo apt update
2. sudo apt install python3-pip
3. python3 -m pip install spyne
4. python3 -m pip install lxml
5. csc proc3.cs
Kolejno uruchamiać:
1. mono proc3.exe
2. python3 proc2.py
3. php proc1.php
